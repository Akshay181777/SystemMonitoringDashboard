# Real-Time System Monitoring Dashboard

A Windows desktop dashboard built using C#, WPF, and LiveCharts2 to monitor system performance in real time.

## Features

- Displays CPU usage
- Displays memory usage
- Displays disk usage
- Displays running process count
- Auto-refreshes every 5 seconds
- Real-time line chart for CPU, memory, and disk utilization
- Clean dashboard UI built using WPF

## Tech Stack

- C#
- WPF
- .NET 8
- LiveCharts2
- System.Diagnostics
- System.Management

## How to Run

1. Open the project in Visual Studio.
2. Restore NuGet packages.
3. Build the solution.
4. Run the application.

## Use Case

This project demonstrates how desktop applications can monitor local system health and visualize resource utilization in real time.
